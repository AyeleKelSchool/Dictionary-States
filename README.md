# Dictionary-States
This program introduces the usage of dictionaries, with the usage of functions.
The dictionary titled popData contains the following key-value pairs: states and it population. 
While the function  retrieve_pop() that takes in user given state and the dictionary itself will loop through the dictionary trying to find the given State(key) and return it value( population) back to the user.
The program run until user specifically enter "no" after being prompted on whether they would like to search for another state population.
